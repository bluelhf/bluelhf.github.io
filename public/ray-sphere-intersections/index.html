Finding ray—sphere intersections

This webpage documents my process for deriving the equations that are used to find the intersections between a ray and a sphere, if those intersections exist. It's mostly rendered through Mafynetti's LaTeX editor, but I've also added my own finishing touches for more smooth inlining of equations in text, as well as this header and some links.
